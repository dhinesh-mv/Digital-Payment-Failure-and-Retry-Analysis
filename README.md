# Digital Payment Failure and Retry Analysis

## Internship Project

This project analyzes digital payment transactions to understand the reasons behind payment failures and customer retry behaviour. The analysis was performed using Python for data preprocessing and exploratory data analysis, and Power BI for interactive dashboard creation.

---

## Project Objective

- Analyze payment success and failure rates
- Study retry behaviour after failed transactions
- Identify common failure reasons
- Visualize payment trends
- Generate business insights

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Microsoft Power BI
- Microsoft Excel

---

## Dataset

The dataset contains digital payment transaction records including:

- Transaction Status
- Retry Attempts
- Payment Method
- Failure Reason
- Transaction Amount
- Date and Time

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. Business Insights
6. Power BI Dashboard

---

## Key Insights

- Identified the payment methods with the highest failure rates.
- Analyzed retry patterns after failed transactions.
- Evaluated transaction success percentages.
- Visualized trends across payment channels.
- Provided recommendations to improve transaction success rates.
